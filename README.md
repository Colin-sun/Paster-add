### 项目说明：
Paster++是对windows的剪贴板的扩展，1.0版提供了剪贴板历史记录，黏贴快捷键，以及json文件的导出。

---
### 项目文件树：
```
├─keys_set.ini #键盘配置
├─key_board.py # 键盘监听文件
├─main.py # 主程序
├─h_save #历史记录保存路径
└─__pycache__
      ├─key_board.cpython-37.pyc
```
